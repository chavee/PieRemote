# PieRemote

A cloud-controlled IR remote powered by NETPIE.io.

**Parts**
- NodeMCU or any ESP8266 dev boards
- IR 38kHz receiver e.g. TSOP4838
- IR LED e.g. TSAL6400, TSAL7400
- Color LED
- 1kΩ resistor
- 470Ω resistor
- 2N 2222 transistor
- Jumper wires
- Breadboard
